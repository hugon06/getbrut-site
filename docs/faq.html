FAQ

#Is Brut free?

The app is free. You pay the providers directly for generation, on your own keys, at their price — Brut takes no cut. Signing in is required (it's how we count seats during the beta), but the free tier is never gated behind payment.

#Where do my API keys go?

Into your OS keychain (Windows Credential Manager). They're used only for direct calls from your machine to each provider, are never written into a graph, never sent to us, and never appear in a .brutpack.

#Does Brut see my prompts, images or renders?

No. Everything you make stays on your disk (or your NAS). Brut has no server in the generation path. The only thing that ever leaves your machine is a tiny sign-in heartbeat — version/OS/install-id, no content.

#Why do I have to sign in?

To count seats during the beta, nothing more. It's required, but it never bricks you: once you've signed in, a 30-day offline grace keeps you working through flights and outages, and if the sign-in server is unreachable a "continue for now" option appears.

#My first generation returned a 429. Why?

Almost always billing isn't enabled on that provider's key — free tiers have zero generation quota, so the first call fails. Enable billing in the provider's console. A first-call 429 is billing, not throttling.

#An image edit came back blank / refused.

On Gemini, finishReason: OTHER is a silent content refusal, usually triggered by an input image — a real person's name in facade lettering, a face, a logo — not the prompt. Remove or crop the offending image. Brut never auto-retries it (it's deterministic).

#Do I ever pay twice for the same thing?

No. Every node hashes its inputs; unchanged work is served from cache for $0. See Never pay twice.

#Can two people share a project?

Yes — put the project on a NAS and each seat opens the same folder, or pass a .brutpack file (warm caches included). There's no cloud sync by design; collaboration is file/NAS exchange.

#What happens if I kill the engine mid-render?

Video jobs are journaled and reattached on the next engine launch — same job ids, one ledger row, correct price. Paid work survives restarts.

#Which video provider should I use?

For steady Kling use, the subscription is ~3× cheaper than API packs. Veo rides your Gemini key. Seedance does native audio. All quote the cost before you spend.

#How do I see my Kling plan credits?

Pick the Kling (subscription) provider on an Image → Video or Film Shots node — the node shows your live plan balance ("🎟 Plan balance: 5,040 credits · Pro") read from the Kling CLI, and Film Shots repeats it in the paid-run confirmation.

#How do I report a bug or ask for a feature?

The 💬 button in the toolbar — see Install & first run. Diagnostics are opt-in and previewed byte-for-byte before sending.

#Does the website track me?

No. getbrut.app and these docs load nothing from third parties — self-hosted fonts, no analytics, no trackers.
