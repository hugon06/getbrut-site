OpenRouter

OpenRouter is Brut's text-route aggregator: one key that can run the planning side of Brut — Refine, the Multi-Shot planner, the cinema crew (Scenarist, Artistic Director, DoP, Script Supervisor) and Text→Text — when you hold no native Gemini / OpenAI / Anthropic key. Like every route, it adds no models to the menu and is always visible: routed planning nodes show ⇄ runs via OpenRouter.

Every text model in Brut's catalog is covered, at pass-through provider list rates — the same per-token prices as the native APIs (OpenRouter's revenue is a ~5.5% fee when you buy credits, not a token markup). Vision-grounded planning works (agents that look at your stills), and the GPT-5.6 reasoning dial rides along.

#Get a key

Create a key at openrouter.ai/keys (sk-or-…), buy a few dollars of credits, and paste it into ⚙ Keys → OpenRouter (or set OPENROUTER_API_KEY).

#The fine print

  • Your native key always wins. Add a Gemini/OpenAI/Anthropic key later and those models bill direct again automatically.
  • A fal.ai key can also run text (fal resells the same router) — but fal's text lane takes no images and no reasoning dial, so OpenRouter outranks it when both keys exist. Vision-grounded calls on a fal-only seat fail with a clear message.
  • Marcel still needs a native text key for now. The companion's streaming chat and prompt-cache lanes don't route yet — this is the one honest gap in the single-key story.
  • Every routed planning call lands in the spend ledger with provider-reported token counts, marked "via openrouter".
