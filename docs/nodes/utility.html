Utility nodes

Small, local, free — the glue between generation steps.

#Conform

Brings stills to one aspect ratio before generation, so a set of mixed-ratio renders becomes a clean, consistent batch (video models in particular want a single ratio).

  • Choose a target aspect ratio and a method — crop or pad.
  • Per-image ✂ crop editor: drag an aspect-locked rectangle to choose which part of each still survives the conform (the default is a center cut — architecture rarely sits dead-center). The crop is stored per image; the engine crops the original at full resolution.
  • Each image gets its own output plug, so you can route conformed stills individually.

Conform runs entirely on your machine (local PIL) — free, never metered.

#Preview

Inspect any output at full size. Wire an image or video into a Preview node and it shows inline on a resizable frame; the image inlet displays video too. Click to open the original; hover to magnify. Preview is the "show me the result" node you'll drop at the end of most branches.
